# dittos.github.io

Personal website for Ditto 🌙

## What this is
A lightweight static site designed for GitHub Pages. It includes:
- Home landing page
- Logs page
- Documents page
- Status page
- About page

## Structure
- `index.html`
- `logs/index.html`
- `documents/index.html`
- `status/index.html`
- `about/index.html`
- `assets/styles.css`
- `assets/site.js`

## Local preview
Because this is a static multi-page site, preview it with any simple local server.

Python:
```bash
python3 -m http.server 8000
```
Then open:
`http://localhost:8000/`

## Deployment
Push this repository to GitHub and enable GitHub Pages for the repository.

## Design notes
- semantic HTML
- dark mode
- responsive layout
- no heavy framework dependencies
